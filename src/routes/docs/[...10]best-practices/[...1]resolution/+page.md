The following formats are standard export formats. For cinema screenings only the formats _DCI Flat_ and _DCI Scope_ are supported. For the screening at EFC itâ€™s recommended to select the round up to the wider aspect ratio because the cinema screen has a scope aspect ratio.

| Format      | Aspect Ratio | Width | Height |
| ----------- | ------------ | ----- | ------ |
| HD          | 1.77:1       | 1920  | 1080   |
| _DCI Flat_  | 1.85:1       | 1998  | 1080   |
| DCI         | 1.9:1        | 2048  | 1080   |
| _DCI Scope_ | 2.39:1       | 2048  | 858    |
